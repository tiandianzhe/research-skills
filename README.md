# Research Skills Collection

A curated collection of WorkBuddy skills for academic research, scientific writing, and web development.

## Skills Included

### 📝 Academic Writing & Research
| Skill | Description |
|-------|-------------|
| `literature-review` | 系统性文献综述，结构化检索与综合分析 |
| `literature-search-strategies` | 高效文献检索策略，数据库使用技巧 |
| `peer-review` | 同行评审指南，审稿意见撰写规范 |
| `scientific-critical-thinking` | 批判性思维框架，科研论证逻辑 |
| `scientific-brainstorming` | 研究创新方法，头脑风暴与假设生成 |
| `imrad-structure` | IMRaD 论文结构规范（Introduction, Methods, Results, Discussion） |
| `reporting_standards` | 科研报告标准，CONSORT/PRISMA/STROBE 等 |
| `figures-tables` | 科研图表设计规范，数据可视化 |
| `database-strategies` | 数据库查询优化，数据管理策略 |
| `humanizer-zh` | 去除 AI 写作痕迹，使文本更自然 |

### 🎨 Web Design & Development
| Skill | Description |
|-------|-------------|
| `frontend-design` | 网站设计与 Web 应用 UI 开发 |
| `web-artifacts-builder` | 快速构建前端小应用、交互 Demo |
| `canvas-design` | 网页动画、Canvas 可视化图形开发 |
| `theme-factory` | 网站换风格、UI 风格统一化 |
| `webapp-testing` | 测试网站功能与交互，自动化测试 |

## Installation

Place each skill folder into your WorkBuddy skills directory:
```
~/.workbuddy/skills/<skill-name>/
```

## Usage

In WorkBuddy, reference a skill with:
```
@skill://<skill-name>
```

Or load it in conversation by mentioning the skill name.
